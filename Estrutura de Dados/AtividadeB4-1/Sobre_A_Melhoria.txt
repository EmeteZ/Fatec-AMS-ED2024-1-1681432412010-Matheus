No código ajustado, eu tornei o Bubble Sort mais eficiente ao incluir uma verificação que interrompe as iterações quando o array já está ordenado, evitando trabalho desnecessário. Separei a lógica de troca em uma função própria, o que deixou o código mais organizado. Também adicionei a opção de o usuário inserir os valores do array e incluí uma medição do tempo de execução para ver o desempenho. Além disso, fiz uma verificação simples para garantir que o tamanho do array seja válido e limpei os comentários desnecessários, deixando o código mais direto.
